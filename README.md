# Hosts Manage

hosts文件管理工具，同时支持基于hosts文件启动DNS代理，共享给其它设备或其他人使用。

## 功能
1. [x] 支持多组hosts配置，多组之间开启后且关系组合。
2. [x] 支持DNS代理，A记录请求会匹配到hosts文件中设置的ip域名映射。
3. [x] 支持获取github等开发常用域名socks5代理，可能会加快访问速度。
