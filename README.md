# hosts_manage

hosts文件管理，可选择使用dns方式

## 备注

需要替换的域名
```
github.com
codeload.github.com
assets-cdn.github.com
github.global.ssl.fastly.net
stackoverflow.com
stackexchange.com
superuser.com
```
